# XRWay’26 Image Guidelines

## 대문이미지 / hero image

- Path: `assets/images/hero.jpg`
- Recommended resolution: **2400 × 1350 px**
- Aspect ratio: **16:9**
- File type: JPG or WebP, but keep the filename `hero.jpg` unless you also update `index.html`
- Design note: the title text overlays the left side, so keep important visual content centered or on the right.

## Social preview image

- Path: `assets/images/og-image.jpg`
- Recommended resolution: **1200 × 630 px**
- Used by Open Graph / social media previews.
